# Automotive Battery Data Analysis

This project provides a Python-based tool for analyzing battery measurement data.  
It focuses on monitoring voltage behaviour and identifying trends in battery systems.

The script reads battery data from CSV files and visualizes voltage changes over time.

## Features

- Import battery data from CSV
- Visualization of battery voltage over time
- Simple analysis of battery behaviour
- Engineering-oriented data evaluation

## Technologies

- Python
- pandas
- matplotlib

## Example Data

time,voltage,current,soc  
0,12.4,5,80  
1,12.3,5.1,79  
2,12.2,5.2,78  

## Run the Script

Install dependencies:
pip install pandas matplotlib
Run the script:
python battery_analysis.py
